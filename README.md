# CodeAlpha_Phishing-Awareness-Traning
This training is designed to educate users on identifying, avoiding, and reporting phishing attempts that target developer accounts and repositories. With attackers increasingly using phishing emails, malicious links, and fake GitHub login pages to steal credentials or inject malicious code, it is critical for developers and teams to stay vigilant.

# Phishing Awareness Training Module 

## Project Overview
A cybersecurity awareness project designed to educate users about phishing attacks.

## Objective
To help users:
- Identify phishing emails
- Recognize fake websites
- Understand social engineering tactics

## Contents
- Example phishing email
- Fake login demo
- Prevention checklist
- Awareness presentation slides

## Common Phishing Indicators
- Suspicious links
- Urgent tone
- Spelling mistakes
- Fake domains

## Prevention Tips
- Verify sender email
- Hover before clicking links
- Enable 2FA
- Report suspicious emails

## What I Learned
- Social engineering concepts
- Email spoofing techniques
- User awareness importance
